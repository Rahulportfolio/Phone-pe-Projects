Phone Pe Analytics Dashboard
Project Overview

The Phone Pe Analytics Dashboard is an interactive Power BI project designed to analyze key business metrics across multiple Phone Pe services, including Insurance, Loans, Money Transfer, and Recharge & Bills. The dashboard provides a comprehensive view of transaction performance, user activity, payment success rates, and service utilization.

Project Purpose

The purpose of this project is to transform raw transaction data into meaningful insights that help understand customer behavior, transaction trends, service performance, and operational efficiency.

Technologies Used
Power BI
PostgreSQL
Microsoft Excel
Power Query
DAX (Data Analysis Expressions)
Data Source
Synthetic Phone Pe dataset generated with ChatGPT for learning and portfolio purposes.
Data includes Insurance, Loans, Money Transfer, and Recharge & Bills transactions.
Dashboard Pages
1. Executive Overview

Provides a high-level summary of key metrics across all services.

2. Insurance Analysis

Analyzes insurance transactions, payment status, user activity, and transaction trends.

3. Loans Analysis

Tracks loan-related transactions, approval status, and performance metrics.

4. Money Transfer Analysis

Monitors transfer volume, transaction success rates, transfer types, and failure reasons.

5. Recharge & Bills Analysis

Evaluates recharge categories, bill payments, transaction trends, and payment performance.

Business Problem

Digital payment platforms generate large volumes of transaction data. Without proper analysis, it becomes difficult to monitor performance, identify transaction failures, understand user behavior, and track service growth.

Dashboard Goals
Monitor overall transaction performance.
Analyze user engagement across services.
Track payment success and failure rates.
Identify common transaction failure reasons.
Understand monthly transaction trends.
Compare service performance across different categories.
